🍕 Bella Italia - Do Zero 

Este site não é um template pronto. Foi o projeto onde bati cabeça para entender como o front-end funciona na prática, conectando estrutura, estilo e lógica real.

A ideia foi criar um cardápio digital direto ao ponto: o cliente olha, escolhe o sabor e já recebe as intruções para o pedido no canal de atendimento.

🔥 Como foi feito?
Esquece os termos de dicionário. Aqui o que mandou foi:

HTML5 (A Base): Estrutura limpa para as pizzas aparecerem onde devem, com foco em organização semântica.

CSS3 (O Visual): Usei Flexbox e Grid para garantir que o layout fique firme tanto no PC quanto no telemóvel.

JavaScript (A Mágica): Criei um script que lê a minha lista de sabores e monta o catálogo sozinho, além de controlar o modal de pedidos que aparece no clique.

Git Bash (Linha de Comando): Nada de arrastar ficheiros. O projeto foi todo gerido e subido via terminal para o GitHub.

🧠 O aprendizado real
O maior desafio não foi escolher a cor do botão, mas sim fazer o JavaScript conversar com o HTML e garantir que os caminhos das imagens estivessem certos para não quebrar no deploy. Ver o git push dar certo e o site subir foi a melhor parte.
